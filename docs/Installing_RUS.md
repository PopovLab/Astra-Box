1. Установить Python 3.10 и выше

2. Установить poetry:
   
   ```
    pip install poetry
   ```

3. Клонировать репозиторий
   
   ```
   git clone git@github.com:temper8/Astra-Box.git
   ```

4. Инициализировать пакеты с помощью poetry
   
   ```
    poetry install
   ```

5. Запустить:
   
   ```
    run.bat
   ```
